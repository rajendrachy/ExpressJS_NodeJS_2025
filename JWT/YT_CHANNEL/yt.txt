----------1. What is JWT?-----------
JWT = JSON Web Token.
It’s a compact, secure way to transmit user identity + claims between client and server.

Commonly used in authentication and authorization.
Self-contained → all info is inside the token.







---------2. Structure of JWT---------
A JWT has 3 parts, separated by .
-> header.payload.signature
